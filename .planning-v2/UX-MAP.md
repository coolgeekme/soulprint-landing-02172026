## Technical Stack

| Layer | Service | Purpose |
|-------|---------|---------|
| Frontend | Next.js + Vercel | UI, SSR |
| Auth | Supabase Auth | Email, Google, SMS |
| Database | Supabase Postgres | User profiles, billing |
| Memory | Mem0 Cloud | Chat history, facts, retrieval |
| Voice | Deepgram API | Transcription, cadence analysis |
| LLM | OpenAI / Anthropic | Chat generation |
| Storage | Cloudinary | Voice recordings |
